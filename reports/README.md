# reports/

Salidas generadas: métricas, tablas y figuras producidas en runtime.

- `figures/` — gráficos que exporta la app o los notebooks. **No se versionan**:
  se regeneran y pueden mostrar datos personales.
- Los CSV de resultados (p. ej. `cv_results.csv`) tampoco se versionan.

No confundir con `assets/`, que **sí** se versiona: ahí van las capturas
curadas que ilustran el README.
